# Project Objectives

## Objetivo General

Construir una simulación progresiva que conecte dinámica neuronal spiking con una salida muscular simplificada e interpretable.

## Objetivos Específicos

- Implementar una neurona Izhikevich individual.
- Explorar respuestas a corrientes constantes, pulsos, rampas, ruido y señales sinusoidales.
- Simular un pool de motoneuronas idealizadas.
- Convertir spikes en una señal de fuerza muscular mediante un modelo twitch.
- Agregar inhibición recurrente tipo Renshaw.
- Crear una visualización 2D simple de contracción muscular o movimiento.

## Alcance Mínimo

- Simulación reproducible de una neurona Izhikevich.
- Pool motor básico con actividad poblacional.
- Conversión de spikes a fuerza simplificada.
- Gráficos claros para voltaje, raster, fuerza y actividad poblacional.
- Documentación suficiente para explicar el flujo del proyecto.

## Extensiones Posibles

- Heterogeneidad más realista en el pool de motoneuronas.
- Modelos de fatiga muscular.
- Delays sinápticos e inhibición con dinámica temporal más detallada.
- Modelo biomecánico 2D de brazo o articulación.
- Interfaz interactiva con sliders para explorar parámetros.
- Inhibición recíproca entre sistemas agonista y antagonista.
